+ Tree
+ Binary Search Tree
+ Heap
+ Graph
+ Greedy 
+ Dynamic Programming
+ Backtracking
+ Graph (Advanced)
+ Trie
+ Segment-Tree
+ Disjoint set

All the problems and exercises associated with each of the topics are solved here
